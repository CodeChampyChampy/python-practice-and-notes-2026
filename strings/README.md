==============================

Python Strings — Exercises

This folder contains a set of Python string exercises designed to practice
real-world text manipulation using built-in string methods.

Each exercise is implemented in its own file.
Some implementations intentionally keep early mistakes for learning purposes.

--------------------------------
Tier 1 — Must-Know (Daily Use)
--------------------------------

Exercise 1 — Clean User Input
Goal:
Remove leading/trailing spaces and normalize text to lowercase.

Input:
"   HeLLo WoRLd   "

Expected Output:
"hello world"

--------------------------------

Exercise 2 — Validate File Type
Goal:
Check whether a filename is a CSV file (case-insensitive).

Input:
"REPORT.CSV "

Expected Output:
True

--------------------------------

Exercise 3 — Hide Sensitive Information
Goal:
Replace the word "password" with "******" in a log message.

Input:
"User entered password incorrectly"

Expected Output:
"User entered ****** incorrectly"

--------------------------------

Exercise 4 — Split a Sentence
Goal:
Split a sentence into individual words.

Input:
"Python makes string handling easy"

Expected Output:
["Python", "makes", "string", "handling", "easy"]

--------------------------------

Exercise 5 — Build a File Path
Goal:
Join path parts using "/" as separator.

Input:
["usr", "local", "bin"]

Expected Output:
"usr/local/bin"

--------------------------------

Exercise 6 — Validate Command Prefix
Goal:
Check whether a command starts with "sudo".

Input:
"sudo apt update"

Expected Output:
True

--------------------------------
Tier 2 — Very Common (Weekly Use)
--------------------------------

Exercise 7 — Locate a Keyword
Goal:
Find the index of the word "error".

Input:
"System error occurred"

Expected Output:
7

--------------------------------

Exercise 8 — Count Character Occurrence
Goal:
Count how many times the letter "o" appears.

Input:
"hello world"

Expected Output:
2

--------------------------------

Exercise 9 — Format a Title
Goal:
Convert text to title case.

Input:
"python string methods"

Expected Output:
"Python String Methods"

--------------------------------

Exercise 10 — Capitalize a Sentence
Goal:
Capitalize only the first character of the sentence.

Input:
"welcome to python"

Expected Output:
"Welcome to python"

--------------------------------
Bonus
--------------------------------

Exercise 11 — Normalize and Validate Email
Goal:
Clean an email and check if it is a Gmail address.

Input:
"  USER@GMAIL.COM "

Expected Output:
True
